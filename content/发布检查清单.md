# 发布检查清单

## Syncer 发布（推荐）

- [ ] 确认要发布的笔记**不是** `个人博客/`、`.obsidian/`、`templates/`、`drafts/`
- [ ] 运行 `npm run prepublish:check -- "目标笔记.md"`（可选但推荐）
- [ ] Obsidian：`Quartz Syncer: Mark` → `Quartz Syncer: Publish`
- [ ] GitHub Actions **Deploy to GitHub Pages** 成功
- [ ] 浏览器刷新博客 URL 确认页面更新

## 本地预检（含 TikZ / 公式）

- [ ] 确认 TikZ 代码块是 ` ```tikz `
- [ ] `npm run publish:note:dry-run -- "目标笔记.md"`
- [ ] `npm run check:publish`
- [ ] `npm run dev` 本地预览

## 一次性 / 环境

- [ ] `npm run verify:pipeline` 通过（baseUrl、workflow、黑名单）
- [ ] Syncer **Test connection** 通过
- [ ] 详见 [SYNCER-SETUP-CHECKLIST.md](../docs/SYNCER-SETUP-CHECKLIST.md)

## 常用命令

```bash
npm run verify:pipeline
npm run prepublish:check -- "测试-渲染.md"
npm run publish:note:dry-run -- "测试-渲染.md"
npm run check:publish
```
